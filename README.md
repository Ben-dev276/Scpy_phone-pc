# Scpy_phone-pc
📱 Affichage de l’écran Android via USB (Batch Scripts)

Ce projet contient un ensemble de scripts .bat permettant d’afficher et de contrôler l’écran de votre téléphone directement depuis votre PC via une connexion USB.

⚙️ Prérequis :

Activer le mode développeur sur votre téléphone
Activer le débogage USB
Connecter le téléphone à l’ordinateur avec un câble USB

🚀 Fonctionnalités :

Affichage en temps réel de l’écran du téléphone sur PC
Possibilité d’interagir avec le téléphone depuis l’ordinateur
Installation rapide via scripts automatisés

💡 Cas d’utilisation :

Utiliser un téléphone avec écran cassé ou inutilisable
Contrôler son téléphone depuis un PC pour plus de confort
Profiter de la puissance et du grand écran de son ordinateur

📝 Ce projet est idéal pour le dépannage, la démonstration ou simplement améliorer votre expérience mobile depuis un ordinateur.
